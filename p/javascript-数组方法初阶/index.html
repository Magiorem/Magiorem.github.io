javaScript-数组方法(初阶)

数组方法应用

Oct 30, 2023
阅读时长: 4 分钟

数组中的初阶方法

数组（Array）数组也是一种复合数据类型，在数组中可以存储多个不同类型的数据；数组中存储的是有序的数据

const arr = new Array()
const arr2 = []     //数组字面量

数组的类型也是object

console.log(typeof arr)		//object

数组的遍历（for-of）

const arr = ["aa", "hh", "123", "genshin"]

for (let i=0; i<arr.length; i++) {
  console.log(arr[i]);
}

for-of语句可以遍历可迭代对象（但只可以顺序进行）

const arr = ["aa", "hh", "123", "genshin"]
for (let value of arr) {
    console.log(value)
}

for (let value of "hello") {
    console.log(value);
}
//字符串也是可迭代对象，h,e,l,l,o

数组的方法（非破坏性方法）

Array.isArray() 检查一个对象是否是数组

Array.isArray(arr)

const arr = ["aa", "hh", "123", "genshin"]

console.log(Array.isArray(arr));	//true

at() 根据索引获得元素（可以负索引）

const arr = ["aa", "hh", "123", "genshin"]

console.log(arr.at(-1))		//genshin

concat() 连接两个数组

const arr = ["aa", "hh", "123", "genshin"]
const arr2 = [1, 2]
console.log(arr.concat(arr2));	//['aa', 'hh', '123', 'genshin', 1, 2]
																//arr不会有任何变化（非破坏性）

indexOf() 第一次出现的索引

lastIndexOf()可以查最后一次出现的位置如果没有则返回-1

const arr = ["aa", "hh", "123", "genshin"]

console.log(arr.indexOf("genshin"));

join() 数组转字符串

参数可以指定一个字符串作为连接符

const arr = ["aa", "hh", "123", "genshin"]

console.log(arr.join()); //aa,hh,123,genshin
console.log(arr.join("-"));//aa-hh-123-genshin

slice() 截取数组

前闭后开第二个参数可以不写，不写则截取到最后

const arr = ["aa", "hh", "123", "genshin"]

const result = arr.slice(0, 2)
console.log(result)			//['aa', 'hh']
console.log(arr.slice(1));	//['hh', '123', 'genshin']
arr.slice()	//可以获得一个一模一样的数组，浅复制

数组的复制

简单的变量赋值不是复制！！没有产生新的对象！

const arr = ["aa", "hh", "123", "genshin"]
const arr2 = arr
arr2[1] = "new"
console.log(arr)   // ['aa', 'new', '123', 'genshin']

console.log(arr2 === arr)   //true

arr和arr2都指向同一个地址，同一个对象

slice() 浅复制

slice不加参数，可以对数组进行一次浅复制，会产生一个新的数组对象

const arr = ["aa", "hh", "123", "genshin"]
const arr2 = arr.slice()
arr2[1] = "new"
console.log(arr)   // ['aa', 'hh', '123', 'genshin']

console.log(arr2 === arr)   //false

浅拷贝和深拷贝

浅拷贝：通常对象的拷贝都是浅拷贝，浅拷贝顾名思义，只对对象的浅层进行复制（只复制一层），如果对象中存储的数据是原始值，那么拷贝的深浅都不重要。浅拷贝只会对对象本身进行复制，不会对对象中的属性或其他东西进行复制

const arr = [{name: "by"}, {name: "haha"}]
const arr2 = arr.slice()

console.log(arr === arr2)				//false
console.log(arr[0] === arr2[0])	//true

!深拷贝：不仅复制对象本身，还复制对象中的属性和元素；因为性能问题，通常情况下我们不用深拷贝

const arr3 = structuredClone(arr)

… 展开运算符

[…arr] 展开运算符可以把一个数组的元素展开到另一个数组中或者作为函数的参数传递（这是一种浅复制）

const arr = ["哈哈", "嘻嘻", "啦啦"]
const arr2 = [...arr]
console.log(arr === arr2)   //false

const arr3 = ["耶耶", ...arr, "呜呜"]

function sum(a, b, c) {
  return a + b + c
}
const arr = [1, 2, 3]
let result = sum(...arr)
console.log(result)		//6

Object.assign()对象的复制

Object.assign(目标对象，被复制的对象)将被复制对象中的属性复制给目标对象，并将目标对象返回

const obj = {name: 'by', age: 18}
const obj2 = Object.assign({}, obj)
console.log(obj2 === obj)   //false

const obj3 = {address: '苏州大学'}
Object.assign(obj3, obj)
console.log(obj3)   //{address: '苏州大学', name: 'by', age: 18}

const obj4 = {...obj}	//也可以使用展开运算符来浅复制
console.log(obj4)

数组的方法（破坏性方法）

push() pop()

push()：向数组的末尾添加一个或多个元素，并且返回数组长度pop()：删除并返回最后一个元素

unshift() shift()

unshift()：向数组开头添加一个或多个元素，并返回新的长度shift()：删除并返回第一个元素

splice()

可以删除、添加、插入、替换

//删除
let arr = ["孙悟空", "猪八戒", "沙和尚", "唐僧"]
arr.splice(1, 2)    //删除的起始位置，删除的数量
console.log(arr)		// ['孙悟空', '唐僧']

//替换
let arr = ["孙悟空", "猪八戒", "沙和尚", "唐僧"]
arr.splice(1, 2, "牛魔王", "红孩儿")  //删除的起始位置，删除的数量，插入的元素（在起始位置插入）
console.log(arr)    // ['孙悟空', '牛魔王', '红孩儿', '唐僧']

//添加
let arr = ["孙悟空", "猪八戒", "沙和尚", "唐僧"]
arr.splice(1, 0, "牛魔王", "红孩儿")
console.log(arr)//['孙悟空', '牛魔王', '红孩儿', '猪八戒', '沙和尚', '唐僧']

数组去重-练习

const arr = [1, 2, 1, 3, 2, 2, 4, 5, 5, 6, 7]

for (let i=0; i<arr.length; i++) {
  const index = arr.indexOf(arr[i], i+1)
  if (index !== -1) {
    arr.splice(index, 1)
    i--
  }
}
console.log(arr)

const arr = [1, 2, 1, 3, 2, 2, 4, 5, 5, 6, 7]

const newArr = []
for (let ele of arr) {
  if (newArr.indexOf(ele) === -1) {
    newArr.push(ele)
  }
}
console.log(newArr)
